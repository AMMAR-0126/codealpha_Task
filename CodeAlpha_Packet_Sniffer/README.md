# CodeAlpha Packet Sniffer

## 📌 Overview
A Python-based **Packet Sniffer** tool that captures and analyzes network packets in real-time. Built as part of the **CodeAlpha Internship** program.

## 🛠️ Tech Stack
- **Language:** Python
- **Library:** Scapy / Socket

## 📂 Files
| File | Description |
|------|-------------|
| `packet_sniffer.py` | Main script to capture and display network packets |

## 🚀 How to Run
```bash
# Install dependencies
pip install scapy

# Run the sniffer (requires root/admin)
sudo python3 packet_sniffer.py
```

## ⚙️ Features
- Capture live network traffic
- Display source & destination IP addresses
- Show protocol type (TCP, UDP, ICMP)
- Real-time packet analysis

## ⚠️ Disclaimer
This tool is for **educational purposes only**. Use only on networks you own or have permission to monitor.
